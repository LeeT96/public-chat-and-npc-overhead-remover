# Public Chat And NPC Overhead Chat Remover
Removes Selected Automatic Public Chat Entries and NPC Overhead Chat Messages
<br>
Took some code from "**Chop Chop Stop**" after installing it and realising it no longer worked.
<br>
<br>
This plugin currently works based on a selection of NPCs using checkboxes; however, I am currently working on allowing a list of NPC IDs to be provided instead so that any new, or missed, NPCs can be added without needing the plugin to be updated.

# Manual List of NPC IDs
_Some NPCs have multiple IDs, therefore you are provided the option to use a checkbox to mute pre-defined NPCs that may have multiple IDs, as well as having the option to provide a list of your own NPCs to mute using their NPC ID.
There is the option to show NPC IDs above their head to save you looking it up online._
